Why MessagePack is Faster

  

    

      
JSON: Character scanning

      

        
Reading string "Alice Nakamura":

        

          
1. Read " (opening quote)

          
2. Read 'A' -- is it \ or "? No, continue

          
3. Read 'l' -- is it \ or "? No, continue

          
4. ... repeat for every character ...

          
5. Read " (closing quote after 14 checks)

        

        
14 branch decisions for a 14-character string.

      

    

    

      
MsgPack: Length-prefix jump

      

        
Reading string "Alice Nakamura":

        

          
1. Read 0xae = fixstr, length 14

          
2. Copy next 14 bytes into buffer

          
3. Done.

        

        
1 type check + 1 memcpy. No scanning.
