Wire Format Breakdown

00 ← magic byte (0x00)
00 00 00 02 ← schema ID = 2 (big-endian uint32)
10 6f 72 64 30 30 30 30 32 ... ← Avro payload (schemaless, 671 bytes)
Total overhead: 5 bytes (magic + schema ID). The consumer uses the schema ID
to fetch the writer's schema from the registry, then resolves it with its own reader schema.
